# **User Guide – Map View / Homescreen**

Beim Start der Anwendung wird automatisch die **Map View (Homescreen)** geladen.
Dieser dient als zentrale Übersicht und ermöglicht die Navigation zu allen Funktionen der App.

---

## **Homescreen – Übersicht & Funktionen**

### **1. Menü (oben rechts)**

* Oben rechts befindet sich ein Button mit **vier Linien**.

* Beim Anklicken öffnet sich eine Liste aller verfügbaren Aktivitäten:

  * Nach **Entfernung sortiert**
  * Untereinander dargestellt

* Im oberen Bereich stehen **Filteroptionen** zur Verfügung:

  * Datum
  * Barrierefreiheit
  * Kategorie/Tag

* Über den **Karten-Button oben rechts** oder die Navigationsleiste gelangt man zurück zur Map View.

---

### **2. Radius-Anzeige**

* Die blaue Leiste mit der Beschriftung **„Radius: X km“** zeigt den aktuell eingestellten Suchradius.
* Dieser bestimmt, welche Events auf der Karte angezeigt werden.

---

### **3. Event-Karten**

* Die rechteckigen Karten stehen jeweils für eine Aktivität.

* Angezeigt werden:

  * Entfernung
  * Anbieter
  * Datum
  * Preis (falls vorhanden)

* Funktionen:

  * **Herz-Button** → Event als Favorit speichern
  * **Distanzanzeige** → Entfernung zum Event

---

## **Navigationsleiste (unten)**

Die Navigationsleiste enthält je nach Nutzerrolle **3 oder 4 Hauptpunkte**:

### **Funktionen:**

* **Fokus**

  * Zentriert die Karte wieder auf den eigenen Standort
  * Hilft, den Überblick nicht zu verlieren

* **Favourites**

  * Zeigt alle favorisierten Aktivitäten an

* **Home**

  * Führt jederzeit zurück zum Homescreen

* 👤 **My Profile**

  * Öffnet das Benutzerprofil

---

## **Profilbereich**

### **Nicht eingeloggt**

* Es wird ein **Login-Button** angezeigt
* Nutzer können sich:

  * einloggen oder
  * registrieren (Name, E-Mail, Passwort)

---

### **Eingeloggt**

Folgende Informationen werden angezeigt:

* Name
* E-Mail
* Rolle
* User-ID

**Weitere Funktionen:**

* Logout-Button → Abmelden
* Liste eigener Events:

  * Alle Details sichtbar
  * Möglichkeit zum Löschen der Events

---

## **Zusätzliche Funktion für Anbieter**

Wenn ein Nutzer als **Anbieter** eingeloggt ist:

* **„Add Event“-Button (ganz rechts)**

  * Ermöglicht das Erstellen eigener Events

* Hinweis:

  * Die **hellblauen Buttons** dienen zum automatischen Ausfüllen mit Testdaten (nur für Testzwecke)
